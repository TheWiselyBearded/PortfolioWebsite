Virtual Keyality

A virtual-reality game that teaches you how to type. The game works by sending phrases into the game from an external server, which is populated through an HTML page or an Amazon Alexa skill. After the phrases have been submitted, the game renders the phrases into the game and the player must type out the corresponding letter as it's flying toward them in order to destroy it and gain points.
